Q1. Write a class which has a function which can take any function as an argument. Execute the function passed as an argument and Print the name of the function to be executed.

Q2. Create a class which has an array with some values. Constitute a function which can eveluate whether a user provided parameter is present in the array using Predicate delegate.

Q3. Write a program which takes two arguments and does some operation. If the value of the two operand is same then throw a custom exception called MyArgumentException 
which passes a message "Two operands cannot be same". Print the message on the caller class.

Q4. Write a program which uses the EventHandler delagate to notify the caller using a custom eventArgs. 
In the custom eventargs you should have your Message and an exception object to notify whether exception occured or not.
